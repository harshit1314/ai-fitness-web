# ⚡ PRIME AI — Elite Fitness & Nutrition Suite

PRIME AI is a high-performance, generative AI fitness companion designed to forge your future self. Built with a modern tech stack and elite-level aesthetics, it provides a personalized, interactive, and data-driven fitness experience.

## 🚀 Key Features

### 🏋️ Intelligent Workout Forging
- **Personalized Planning**: Generates 7-day workout routines tailored to your biometrics, goals, and equipment availability.
- **Dynamic Exercises**: Every plan includes specific set/rep counts with AI-authored performance notes.

### 🥗 Precision Nutrition Engine
- **Macro-Aided Dieting**: High-protein, calorie-matched meal plans based on your dietary preferences (Veg/Non-Veg/Vegan/Keto).
- **Macro Calculator**: Real-time interactive tool to adjust your daily nutritional targets.

### 🧠 Visual Cortex & Virtual Coach
- **AI Image Generation**: Click any exercise or meal to generate a high-fidelity visual representation powered by Pollinations.ai.
- **Voice Synthesis**: Hear your plan read aloud by an AI coach for hands-free gym sessions.
- **AI Chat Assistant**: A dedicated fitness expert available 24/7 to answer your specific nutrition and training questions.

### 📑 Professional Utility
- **PDF Export**: Download your entire plan as a clean, professionally formatted document.
- **Data Persistence**: Automatic saving to `localStorage` ensures your plan is always ready when you return.
- **Mindset Shift**: Daily AI-generated motivation quotes and elite performance tips.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **AI Engine**: Google Gemini 1.5 Flash
- **Styling**: Tailwind CSS, Framer Motion (High-Fidelity Animations)
- **Icons**: Lucide React
- **PDF Engine**: jsPDF
- **Visuals**: Pollinations.ai API

## 🏁 Getting Started

1. **Environment Setup**:
   Copy `.env.example` to `.env.local` and add your keys:
   ```env
   GEMINI_API_KEY=your_key_here
   ELEVENLABS_API_KEY=your_key_here (optional for voice)
   ```

2. **Install & Run**:
   ```bash
   npm install
   npm run dev
   ```

3. **Ignite**:
   Open [http://localhost:3000](http://localhost:3000) and start your transformation.
